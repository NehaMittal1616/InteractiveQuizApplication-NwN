# Interactive Quiz Application

This is an interactive quiz application where users can test their knowledge in Geography and History. Users are prompted to enter their name and then answer five multiple-choice questions randomly selected from an external API. After completing the quiz, users will see their score along with their previous attempts.

## Features
- User can input their name and start the quiz.
- Five random questions are fetched from the Open Trivia Database API.
- Users have 15 seconds to answer each question.
- Immediate feedback on correct/incorrect answers.
- A progress bar to track quiz completion.
- User score and previous attempts are displayed at the end of the quiz.
- Responsive design suitable for both desktop and mobile devices.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)

## Setup Instructions

### Prerequisites
To run the project, you only need a modern web browser (Chrome, Firefox, Safari, etc.) and a text editor (VSCode etc.) for making changes.

### How to Run the Project
1. **Clone the repository** or **download the project** files:
2. **Navigate to the project directory**:
3. **Open the project**:
- You can simply open the `index.html` file in any web browser.
- If you are using VSCode, you can right-click on the `index.html` file and select "Open with Live Server" (if you have the Live Server extension installed).

4. **Start the Quiz**:
- Open the webpage in your browser.
- Enter your name in the text box and click the "Start Quiz" button.
